cd - change directery. ls - list out items in current location. mkdir - make a directory/folder. del (file name without the paranthesis) - removes a file. rmdir - removes a folder. cmd +/- - increases and decreases the command line. touch - creates a file. rm - remove files. git config --global user.name "John Doe". git config --global user.email "johndoe@example.com". git add [file_name] (adds a specific file to the staging area). git add . (adds all files to the staging area). git commit -m “commit message” (this message will describe changes). git push (upload your code to GitHub). git pull (download changes from GitHub). git-bash commands. git clone - clone a repository. git branch- shows all branches that exists. git add- readies files/folders to push to github. git status- checks what files are ready to push. git commit -m "brief description"- commit files and adds a message to it. git push origin branch-name- pushes our code to the following branch. git checkout -b new branch name. git checkout branch-name - switches branches. git pull origin branch-name-pull/download our code from branch. git init- helps us create a repository. git -lsa: it shows all files even the hidden ones. git logs: what has been changed and it shows the notes that you write. git (log, --oneline)- its more of a brief overview. git checkout takes you to a branch. git branch: shows the list of branches. git switch -c mybranch: does the same thing as git checkout -b mybranch. git merge main: you will have a branch open and this merges the main file with branch file.
